# 隨堂答疑

不屬於主課程大綱、但學習過程中問到的問題與回答。

## 命名

`YYYY-MM-DD-短主題.md`，例如：

- `2026-05-13-why-utun-numbering.md`
- `2026-05-20-cloudflare-warp-vs-vps.md`

## 模板

```markdown
# 問題：（一行）

## 背景
（為什麼會問這個？當時學到哪一堂？）

## 回答

## 與大綱的關聯
- 對應到 [Part X.Y](../lessons/...)（如有）
- 衍生到 [Part X.Y](../lessons/...)（將來會深入講）
```
